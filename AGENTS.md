# Agent Instructions

## Co-Writing Workflow

- `/write` — co-write a post: light interview, 3 candidates, saved to `.vertical/gtm/posts/`.
- `/capture` — after publishing: record the final version and mine the draft→final diff into taste rules.
- `.vertical/gtm/TASTE.md` — accumulated voice rules with real before/after evidence. When drafting anything, the calibration pairs in that file outrank the general guidance below.
- `.vertical/gtm/sources.md` — append-only capture for links, quotes, and external references. When Micha shares a link or quote worth keeping, append an entry there (newest on top) without being asked.
- `.vertical/gtm/notes/` — Micha's own longer-form thinking notes (assumptions, whiteboard sessions, memos). One file per note. These are raw material for posts; `/write` may mine them.
- `.vertical/gtm/styles/` — author style packs (`<author>.md`, compressed how-to-write) with eval companions (`<author>-rubric.md`, score a text against the pack). First pack: derek-sivers.
- `.vertical/gtm/essays/` — long-form essay drafts, one file per essay (`YYYY-MM-DD-slug.md`, see TEMPLATE.md). A `Style:` header names the style pack to draft/eval against.
- `.vertical/gtm/transcripts/` — stored transcripts of videos/talks referenced in sources.md.
- Background research on the larger system (eval loops, LLM wiki) lives in `.vertical/gtm/research/`.

## Voice For Content Work

When drafting posts, replies, launch notes, or founder-led content for this repo,
write like a normal person on X or LinkedIn, not like polished marketing.

People value authentic, specific, human-sounding text.

Prefer:
- concrete observations
- working notes
- personal experience
- simple words
- lowercase / low-punctuation style when the user is drafting for X
- sharing what is being built and letting people pull

Avoid:
- polished marketing language
- needy engagement bait
- salesy lines like "curious who else is running into this"
- pitch phrasing
- over-explaining the thesis
- turning every post into a lesson

Default shape:
- say what was noticed
- say what was tried or built
- say the small insight
- stop before it becomes a pitch
