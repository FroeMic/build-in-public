# Sources

Append-only capture for external material: links, posts, quotes, articles,
threads, anything that sparked a thought. Newest entry on top. No filing, no
categories — just append and move on. During `/write`, recent entries here are
raw material for posts.

Entry format (loose — a bare link with one line is fine):

```
## YYYY-MM-DD — short handle
<link>
why: one line on why this is worth keeping
> optional quote worth saving verbatim
```

Two kinds of `why` worth distinguishing when it's easy:
- what it says (an idea to respond to or build on)
- how it's written (a line or shape to learn from — feeds TASTE.md eventually)
